# 🧠🎙️ BrainBerry – Voice-Driven Learning Platform for SLD & Dyslexic Children

> *Learn. Speak. Play.*

BrainBerry is an interactive, voice-driven learning platform designed specifically for children with **Specific Learning Disabilities (SLD)** and **dyslexia**. It focuses on improving learning through **speech-based interaction**, **visual engagement**, and **gamified education**.

---

## 🌟 Key Features

### 🎤 Voice-Driven Learning

* Speech-to-text interaction for hands-free learning
* Encourages active participation through voice commands

### 📚 Interactive Lessons

* Structured modules like:

  * Magic of Words
  * Numbers
  * Golden Words & Sentences
  * Civic Sense

### 🎮 Gamified Experience

* XP system and levels
* Engaging UI designed for kids
* Reward-based progression

### 🧩 Accessibility-Focused Design

* Built specifically for **SLD & Dyslexic learners**
* Visual + auditory learning approach
* Simple and distraction-free interface

### 🔐 Authentication (Firebase)

* Secure login/signup system
* User session handling
* Scalable backend authentication

### 📝 Task & Learning Management

* To-do list for daily learning goals
* Personalized learning tracking (UI-based)

## 🚀 How It Works

1. User interacts via voice 🎤
2. Speech is converted to text
3. Platform processes input
4. Lessons and responses are displayed interactively
5. Progress is visually tracked
